public static bool ToggleGroup(GUIContent title, string property, ToggleMode toggleMode, GroupStyle style, Action<GenericMenu> context, Action content, Material material = null);

Properties

titleThe title shown in the head of the group
propertyThe property in the material which contains if this group is open or closed
toggleModeThe mode how to save the open/close state and what to manipulate
styleThe style used for the group border
contentThe content of the group: anonymous function possible (goes onto the heap) - delegate() {}
materialThe material to use for the group commands
contextThe context menu of the group (menu, groupContent, groupLevel): anonymous function possible (goes onto the heap) - delegate() {}

Returns

bool - If the group is open

Description

Displays a group containing all fields which are used in the content.
